# Installation

## Compatibility

- Vue.js 3.x
- maplibre-gl-js 5x.x

## Install


::: code-group

```sh [npm]
npm install @indoorequal/vue-maplibre-gl maplibre-gl
```

```sh [yarn]
yarn add @indoorequal/vue-maplibre-gl maplibre-gl
```

```sh [pnpm]
pnpm add @indoorequal/vue-maplibre-gl maplibre-gl
```

:::
